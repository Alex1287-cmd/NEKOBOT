### Install
Clone this project

```bash
> pkg install git
> git clone https://github.com/Alex1287-cmd/NEKOBOT
> cd NEKOBOT
```

Install the dependencies:

```bash
> pkg install bash
> bash install.sh
```

### Usage
run the Whatsapp bot

```bash
> node index.js
```
after running it you need to scan the qr
